## Foreword

pdf-fmt is a PDF text extraction utility. It supports features like regex based
filtering and document format conversion (via LibreOffice or pandoc).

### Project Status

The project is currently under active development. There are now pre-built binaries
for all major platforms (Windows x86-64, Linux x86-64, Mac ARM).

For other platforms `uv install` should work.
